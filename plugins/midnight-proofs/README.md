# midnight-proofs

Server-side ZK proof generation, verification, and optimization for Midnight Network.

## Skills

| Skill | Description |
|-------|-------------|
| `proof-generation` | Server-side proof creation |
| `proof-verification` | Verify proofs before transactions |
| `proof-caching` | Cache and reuse proof components |
| `prover-optimization` | Tune prover performance |

## When to Use

- Offloading proof computation from client devices
- Building proof-as-a-service backends
- Optimizing prover performance for production
- Implementing proof caching strategies

## Related Plugins

- `compact-core` - Writing Compact contracts
- `midnight-tooling` - Deployment and calling
- `midnight-dapp` - Frontend proof integration
