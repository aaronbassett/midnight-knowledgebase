# midnight-indexer

Query blockchain data from Midnight Network for building data-driven DApps.

## Skills

| Skill | Description |
|-------|-------------|
| `indexer-service` | Connect to and query the indexer |
| `event-subscriptions` | Real-time WebSocket event streams |

## When to Use

- Querying transaction history
- Fetching account balances
- Reading contract state from the blockchain
- Subscribing to contract events in real-time
- Building data-driven DApp backends
- Monitoring UTXO state changes

## Indexer API Versions

This plugin covers the following Midnight Indexer API versions:

| Version | Status | Notes |
|---------|--------|-------|
| 2.0.0 | Supported | Base functionality |
| 2.1.0 | Supported | Enhanced query filters |
| 2.1.4 | Latest | Performance improvements |

## Related Plugins

- `midnight-tooling` - Deployment and contract calling
- `midnight-dapp` - Frontend data display
- `midnight-proofs` - Proof generation for transactions
